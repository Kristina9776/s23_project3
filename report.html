This project, upon pressing one of the five provided languages, uses an English word or phrase entered into the search box, translates it using Google Translate's API, and returns a list of 20 songs on Spotify that contain the searched term in the title using Spotify's API.

  1. Our website is deployed publicly on github.io https://kristina9776.github.io/s23_project3/
  2. Our website is responsive to mobile and larger viewports. On mobile, the top nav bar containing our language buttons and search turns into a drop down box.
  3. Our website utalizes both the google translate API and the Spotify API.
  4. Our website contains a search bar where a user can type in a word they want to be translated.
  5. Our website displays the song results from searching using the translated words into the DOM as unordered list elements.
  6. Users will have the option to sort their list of songs alphabetically by pressing a sort button.
